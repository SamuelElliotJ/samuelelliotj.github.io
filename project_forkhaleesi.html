Overview

Compel is a vertical slice of a voice-controlled, action game made for a university module. The goal for this project was to create a voice-controlled action game where the player uses their voice to influence the games world and explore how voice-controls can be used for a wide genre of games. From a technical standpoint my goal was to further improve my skills and knowledge base in Unreal Engine 5.
