Side Scrolling Shooter

This is a game i made for a module in my first year of university. I had only used Unity once before for a smaller project and was still learning all the basics.

This side scrolling shooter game is one 3 stage boss fight with a lovecraftian theme.

My role:

I worked on this alone so i did all of the programming and spriting for the game.

Learning Outcomes:

This was my second time making something in Unity and C#, so i had to familiarise myself with the Unity forums and do a lot of research. It took me a while to get everything done but with the framework and my idea it turned out well.
